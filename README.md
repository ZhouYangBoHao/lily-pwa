# 莉莉丝 PWA 💜

一个零依赖的AI聊天PWA,直接用浏览器打开`lily.html`即可使用。

## 使用方法
1. 下载 `lily.html` 到本地
2. 用浏览器打开 (推荐Chrome/Edge)
3. 点右上角⚙️填入你的 MiniMax API Key
4. 开始聊天!

## 功能
- ✅ 文字聊天 (基于 MiniMax LLM)
- ✅ TTS语音回复 (微软/字节声音)
- ✅ 对话历史保存 (localStorage)
- ✅ 移动端友好 (PWA)
- ✅ 完全离线UI (除API调用)

## 系统要求
- 现代浏览器 (Chrome 90+ / Edge 90+ / Safari 14+)
- API Key (从 https://api.minimaxi.com 获取)

## 文件说明
- `lily.html` - 完整的单文件PWA (15KB)
- 不需要任何服务器、CDN、构建工具
